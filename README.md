## Documentation

A rating app based on S3E1 black mirror for codocodile contest.
Developed with GO(Gin + Gorm) and TypeScript(React)


## Run Locally - Frontend

```bash
  cd frontend/
  npm install
  npm run dev
```

## Run Locally - Backend

```bash
  cd backend/
  cd api/
  go mod download
  cp .env.example .env
  go run main.go
```
